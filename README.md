My Claude Code configuration framework.

## Agents

| Agent | Purpose |
|-------|---------|
| code-analyzer | Analyze code for quality and performance |
| code-reviewer | Review code for quality and security |
| problem-analyzer-planner | Analyze problems and create plans |
| task-planner-implementer | Plan and implement tasks |
| task-executor | Execute multiple tasks sequentially |
| commit-message-generator | Generate commit messages |
| pr-generator | Generate pull requests |

## Commands

| Command | Purpose |
|---------|---------|
| /m-analyze | Analyze code files |
| /m-review | Review modified code |
| /m-plan | Create implementation plans |
| /m-implement | Plan and implement tasks |
| /m-execute | Execute multiple tasks |
| /m-commit | Generate commit messages |
| /m-pr | Generate pull requests |
